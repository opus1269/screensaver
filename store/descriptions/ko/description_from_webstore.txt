구글 포토의 사진을 비롯한, 다양한 출처의 사진을 표시할 수 있는 화면보호기.

이 확장 프로그램은 여러 출처의 사진을 슬라이드 쇼로 구성하여 화면보호기로 표시합니다. 지원되는 사진 출처는 다음과 같습니다.

* 내 구글 사진과 앨범들
      참고 : 다른 사람들과 공유하는 앨범은 지원되지 않습니다.
* Chromecast에서 사용한 일부 배경 사진
* Reddit의 다양한 사진들
* Unsplash의 다양한 사진들
* Flickr의 흥미로운 사진들

참고 : 로컬 파일 시스템에 있는 사진은 지원되지 않습니다.

슬라이드 쇼의 모양을 제어하기 위해 사용할 수 있는 옵션은 다음과 같습니다.

* 화면 보호기를 표시하기 위해 컴퓨터가 대기하는 시간
* 사진이 표시되는 시간
* 사진 표시 방식 - (예 : 레터 박스, 화면 채우기)
* 사진 전환 애니메이션 방식 - (예 : 페이드, 스케일 업)
* 얼굴 감지 기능이있는 "Ken Burns"효과
* 화면 보호기, 디스플레이 및 컴퓨터가 켜져있는 시간을 제어하는 스케줄러
* Chrome 창이 전체화면 일 때(예 : YouTube, Netflix) 화면보호기 작동 방지
* 모든 디스플레이에 표시
* 현재 날씨 표시
* 시간 표시

모든 플랫폼에서 훌륭하게 작동하지만 특히 Chromebook에서 유용 할 수 있습니다.
